Percentage calculator

Enter the Order Amount and the Deduction Value. The tool calculates a percentage you can use in the formula Order amount * percentage = Deduction Value.

Tip: The value copied by default is the decimal percentage (for example 0.15) which you can plug directly into Order * percentage. You can also copy the percent form (15%).
